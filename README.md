# Network Intrusion Detection using Machine Learning

This project implements a machine learning pipeline to detect malicious network traffic using the UNSW-NB15 cybersecurity dataset.

The goal is to analyze network traffic features and build classification models capable of identifying cyber attacks and distinguishing them from normal network behavior.

---

## Project Overview

Modern networks generate massive volumes of traffic, making manual monitoring impossible. Intrusion Detection Systems (IDS) help identify suspicious activities and protect infrastructure from attacks.

This project applies machine learning techniques to build an automated intrusion detection system capable of detecting malicious network behavior.

---

## Dataset

Dataset used:

UNSW-NB15 Network Intrusion Dataset

The dataset contains network traffic records with multiple features describing packet properties and connection behavior.

Features include:

- Source and destination information
- Packet statistics
- Protocol types
- Network service indicators
- Attack labels

---

## Project Pipeline

1. Data preprocessing and cleaning
2. Exploratory Data Analysis (EDA)
3. Feature engineering
4. Machine learning model training
5. Model evaluation
6. Visualization of results
